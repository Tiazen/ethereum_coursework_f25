## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ forge create src/UserVault.sol:UserVault --rpc-url http://127.0.0.1:8545 --private-key $PRIVATE_KEY --broadcast
```


### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
